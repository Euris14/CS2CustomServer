# CS2 Custom Server



## Install Dependencies

```bash
pip3 install -r requirements.txt
```

## Usage

```bash
./server.py -c <choice> or ./server.py -h
```

Commands:
```bash
1. ./server.py -c install
2. ./server.py -c update
3. ./server.py -c validate
4. ./server.py -c start
```
